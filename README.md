# Jokes Bapack2 Extension for Visual Studio Code
Enjoy jokes bapak2 while _ngoding_ on visual studio code

## Requirements
- Visual Studio Code
- npm
- axios

## How to Run
- Clone this repo
- ```cd jokes-bapack2```
- ```npm install```
- ```code .```
- Press <kbd>F5</kbd>
- Open command palette using <kbd>CTRL</kbd>+<kbd>Shift</kbd>+<kbd>P</kbd>
- Type ```joke```
- Hit <kbd>Enter</kbd>
